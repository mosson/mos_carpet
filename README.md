# MosCarpet

redcarpetのマークダウンに独自に書式を追加したライブラリ

## Installation



Add this line to your application's Gemfile:

```ruby
gem 'mos_carpet', github: 'https://github.com/mosson/mos_carpet
```

And then execute:

    $ bundle

## Usage

```
MosCarpet::MarkdownProcessor.to_html('hello, world! and youtube(1234)')
```
